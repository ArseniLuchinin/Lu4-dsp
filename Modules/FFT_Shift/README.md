# FFT_Shift

Сдвиг половин спектра (FFT shift).

Меняет местами левую и правую половины спектра для центрирования нулевой частоты. Работает только с полным комплексным спектром.

## Параметры

| Параметр | Тип | Описание |
|---|---|---|
| `fft size` | int | Размер FFT |
| `window size` | int | Размер окна (0 = автоопределение) |

# Decimator

Децимация (понижение частоты дискретизации) комплексного сигнала.

Выбирает каждый N-й семпл. Сохраняет фазу между вызовами для корректной потоковой обработки.

## Параметры

| Параметр | Тип | Описание |
|---|---|---|
| `samples per symbol` | int | Коэффициент децимации (N) |
| `offset` | int | Начальная фаза (0..N-1) |

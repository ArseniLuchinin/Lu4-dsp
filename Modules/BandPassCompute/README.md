# BandPassCompute

Вычисляет коэффициенты полосового FIR-фильтра для вещественного сигнала.

Использует метод окон с окном Хэмминга. Коэффициенты генерируются один раз и передаются в FIR-filter через VirtualTX/VirtualRX.

## Параметры

| Параметр | Тип | Описание |
|---|---|---|
| `sample rate` | int | Частота дискретизации, Гц |
| `filter order` | int | Порядок фильтра (нечётное число) |
| `low cutoff` | float | Нижняя граница полосы, Гц |
| `high cutoff` | float | Верхняя граница полосы, Гц |

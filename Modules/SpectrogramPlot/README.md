# SpectrogramPlot

Генерирует изображение спектроскопии из амплитудного спектра.

Рендерит на GPU, сохраняет в PNG. Поддерживает настройку диапазона частот, dB-диапазона, цветовую карту Parula.

## Параметры

| Параметр | Тип | Описание |
|---|---|---|
| `sample rate` | int | Частота дискретизации, Гц |
| `fft size` | int | Размер FFT |
| `window size` | int | Размер окна (бинов на строку) |
| `hop size` | int | Шаг между окнами |
| `centered spectrum` | bool | Центрированный спектр |
| `freq min` | float | Минимальная частота отображения, Гц |
| `freq max` | float | Максимальная частота отображения, Гц |
| `db min` | float | Минимум dB для нормализации |
| `db max` | float | Максимум dB для нормализации |
| `mask below db` | float | Порог маскировки значений |
| `save path` | string | Путь для сохранения PNG |
| `png compression` | int | Сжатие PNG (0..9) |

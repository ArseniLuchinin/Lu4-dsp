# FileWriter Module

Записывает данные из GPU в бинарный файл.

## Параметры

| Параметр | Тип | Описание |
|----------|-----|----------|
| file name | string | Путь к выходному файлу |
| overwrite | bool | Перезаписать файл, если существует (по умолчанию false) |
| buffer size | size_t | Размер буфера `std::ofstream` в байтах (по умолчанию 8 МБ) |

## Входные данные

Модуль принимает любые данные, реализующие интерфейс `IGpuSignalData` (все GPU-сигналы в проекте).

## Выходные данные

Модуль передает данные дальше без изменений (как `TextFileWriter`).

## Особенности реализации

- Данные копируются с GPU на CPU через **page-locked (pinned) host memory**, что ускоряет `cudaMemcpy`.
- Хост-буфер выделяется один раз и переиспользуется между вызовами `run()`.
- Запись буферизуется; принудительный сброс на диск происходит только при закрытии файла (в деструкторе).

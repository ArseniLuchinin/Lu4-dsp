# CarrierPhaseEstimator

Оценивает фазу несущей из комплексного сигнала.

Суммирует все семплы (CUB reduce) и вычисляет фазу через `atan2`. Результат передаётся по тегу для PhaseRotator.

## Параметры

| Параметр | Тип | Описание |
|---|---|---|
| `phase tag` | string | Тег для передачи фазы (по умолчанию `qpsk_phase`) |

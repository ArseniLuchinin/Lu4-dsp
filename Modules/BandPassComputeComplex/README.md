# BandPassComputeComplex

Вычисляет коэффициенты полосового FIR-фильтра для комплексного сигнала.

Поддерживает выбор боковой полосы: `negative`, `positive` или `both`. Для комплексных коэффициентов формирует фильтр с гетеродинированием.

## Параметры

| Параметр | Тип | Описание |
|---|---|---|
| `sample rate` | int | Частота дискретизации, Гц |
| `filter order` | int | Порядок фильтра (нечётное число) |
| `low cutoff` | float | Нижняя граница полосы, Гц |
| `high cutoff` | float | Верхняя граница полосы, Гц |
| `sideband` | string | Боковая полоса: `negative`, `positive`, `both` |

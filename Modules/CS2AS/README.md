# CS2AS

Преобразует комплексный спектр в амплитудный спектр (дБ).

Формула: `10 * log10(|z|² * norm)`, где `norm = 1/fftSize²` при включённой нормализации.

## Параметры

| Параметр | Тип | Описание |
|---|---|---|
| `fft size` | int | Размер FFT для нормализации |
| `normalize by fft size` | bool | Включить нормализацию (по умолчанию true) |

# RRCCompute

Вычисляет коэффициенты Root Raised Cosine (RRC) фильтра.

Используется для формирования импульсов в QPSK-системах. Коэффициенты передаются в FIR-filter.

## Параметры

| Параметр | Тип | Описание |
|---|---|---|
| `sample rate` | float | Частота дискретизации, Гц |
| `symbol rate` | float | Символьная скорость, символов/с |
| `rolloff` | float | Коэффициент подъёма (0..1, по умолчанию 0.35) |
| `span symbols` | int | Длительность фильтра в символах |
| `samples per symbol` | int | Семплов на символ (опционально) |
| `normalize gain` | bool | Нормализовать усиление (по умолчанию true) |

# FFT

Быстрое преобразование Фурье (FFT) через cuFFT.

Автоматически определяет тип входа (вещественный/комплексный). Поддерживает overlap-add для потоковой обработки.

## Параметры

| Параметр | Тип | Описание |
|---|---|---|
| `fft size` | int | Размер FFT |
| `hop size` | int | Шаг между окнами (по умолчанию = fft size) |

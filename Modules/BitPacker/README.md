# BitPacker

Упаковывает поток битов в байты.

Входные данные — байты, где каждый байт хранит один бит. Выход — компактные байты (8 бит на байт). Поддерживает отбрасывание начальных битов и упаковку хвоста.

## Параметры

| Параметр | Тип | Описание |
|---|---|---|
| `bit order` | string | Порядок бит: `msb-first` |
| `flush tail` | bool | Упаковывать остаток в дополнительный байт |
| `discard leading bits` | int | Сколько начальных битов отбросить |

# FileSrc

Источник данных — читает сигнал из файла.

Использует mmap для эффективного чтения. Читает блоками заданного размера. Поддерживает вещественный и комплексный форматы.

## Параметры

| Параметр | Тип | Описание |
|---|---|---|
| `file name` | string | Путь к файлу |
| `max size` | int | Максимальный размер блока, байты |
| `data type` | string | Формат данных: `float` или `complex` |
